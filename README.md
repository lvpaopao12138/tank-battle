# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas + JavaScript 实现的经典坦克大战游戏，无需任何依赖，直接在浏览器中运行。

## 🎯 游戏特色

- **经典玩法复刻** — 控制坦克、射击敌人、保卫基地
- **三种关卡** — 经典布局、钢墙挑战、迷宫风格
- **四种敌人** — 普通坦克、快速坦克、重型坦克（2HP）、精英坦克
- **可破坏砖墙 & 不可破坏钢墙** — 利用地形策略作战
- **爆炸特效** — 粒子爆炸动画
- **移动端支持** — 触摸操控，随时随地畅玩
- **暂停/继续** — 按 P 键随时暂停

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| `↑ ↓ ← →` / `WASD` | 移动坦克 |
| `空格` / `J` | 发射子弹 |
| `P` | 暂停/继续 |
| `R` | 重新开始 |

### 移动端
触摸屏幕任意位置来控制方向，自动连射。

## 🚀 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/YOUR_USERNAME/tank-battle.git
```

2. 用浏览器打开 `index.html` 即可开始游戏！

或者直接在浏览器中打开：
- 双击 `index.html` 文件
- 或用 Live Server 等工具启动本地服务器

## 🏗️ 技术栈

- 纯 HTML5 Canvas
- 原生 JavaScript (ES6+)
- CSS3 动画与样式
- 零依赖，零构建步骤

## 🎮 游戏规则

1. **保卫基地** — 地图底部的 🦅 标志是你的基地，被摧毁即游戏结束
2. **消灭敌人** — 每关有 20+ 辆敌方坦克，全部消灭即可过关
3. **生命系统** — 初始 3 条命，被击中后短暂无敌保护
4. **得分规则**：
   - 普通坦克：100 分 | 快速坦克：150 分
   - 重型坦克：200 分 | 精英坦克：300 分
   - 摧毁砖墙：5 分
